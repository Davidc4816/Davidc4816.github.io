How to cretae webpage step by step

  1. Create a folder called github website. Inside the github website folder, create two folders. One named CSS and the other named Images.
  2. Then go to brackets and save it into the github website folder, named Index.
  3. Now go to w3school and copy the html basics and paste it into the index folder.
  4. Then name the index folder to 'index.html'. That will allow you to edit things with html.

How to use adobe media encoder

  1. First film a 30 second video
  2. Then move the video to the adobe media encoder
  3. After that change the resolution to what you like and fits.
  4. If you edit the video to what you like then move the video to the github webpage folder.
